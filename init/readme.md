## Code Example

Нижче наведено приклад коду на Python:

```python
# Python program to calculate factorial

def factorial(n):
    """Calculate the factorial of a number."""
    if n == 0:
        return 1
    return n * factorial(n - 1)

# Test the function
number = 5
print(f"The factorial of {number} is {factorial(number)}")
```
